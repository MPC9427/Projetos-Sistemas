iniciar uma área de trabalho (veja também: tutorial de ajuda do git)

clone Clonar um repositório em um novo diretório
init Criar um repositório Git vazio ou reinicializar um existente

trabalhar na alteração atual (veja também: ajuda do git todos os dias)

add  Adicionar conteúdo do arquivo ao índice
mv  Mover ou renomear um arquivo, diretório ou link simbólico
restore  Restaurar arquivos da árvore de trabalho
rm  Remover arquivos da árvore de trabalho e do índice

examinar o histórico e o estado (veja também: revisões da ajuda do git)

bisect     Usar busca binária para encontrar o commit que introduziu um bug
diff       Mostrar mudanças entre commits, commit e árvore de trabalho, etc.
grep       Exibir linhas que correspondem a um padrão
log        Exibir logs de commits
show       Exibir vários tipos de objetos
status     Exibir o status da árvore de trabalho

aumentar, marcar e ajustar seu histórico comum

backfill  Baixar objetos ausentes em um clone parcial
branch    Listar, criar ou excluir branches
commit    Registrar mudanças no repositório
merge     Unir dois ou mais históricos de desenvolvimento
rebase    Reaplicar commits sobre outra base tip
reset     Redefine o HEAD atual para o estado especificado
switch    Troca de branches
tag        Criar, listar, excluir ou verificar um objeto de tag assinado com GPG


olaborar (veja também: fluxos de trabalho de ajuda do git)

buscar  Baixar objetos e referências de outro repositório
puxar   Buscar e integrar com outro repositório ou uma ramificação local
empurrar  Atualizar referências remotas junto com objetos associados
